# 🧠 Challenge Description

## From Data to Insight: Using Open Data

This challenge aims to promote learning how to use **Open Data** through a structured data analysis process, from data preparation and exploration to extracting insights that support decision-making.

Participants will practice skills in **Data Preparation**, **Exploratory Data Analysis (EDA)**, and **Data Visualization** to discover patterns, trends, and key issues hidden in the data. The results should be presented through a **Dashboard or Visualization** that communicates information clearly.

---

# 🎯 Objective

Analyze the provided **Open Data** to discover **meaningful insights**, and present the analysis results through **visualization or a dashboard** that makes the data easy to understand and useful for decision-making.

---

# 📌 Task

Participants must choose **one dataset from the provided datasets** and perform data analysis following these steps.

## 1️⃣ Data Preparation
- Understand the structure of the dataset
- Clean the data (**Data Cleaning**)
- Transform and format the data so it is ready for analysis

## 2️⃣ Exploratory Data Analysis (EDA)
- Explore the characteristics of the dataset
- Analyze trends, patterns, or relationships in the data
- Create graphs or **visualizations** to better understand the data

## 3️⃣ Insight Discovery
- Identify **important or interesting insights**
- Explain the meaning of the discovered results
- Answer the key questions from the selected dataset
